===============
Contents of "/"
===============

---------
Endpoints
---------

``````
\/
``````

Serves the homepage.

``````````
/{ITEM_ID}
``````````

Provides RESTful access to the URL-specified item.

``````````````````
/{ITEM_ID}/chatter
``````````````````

Generates chatter.

````````````````````
/{ITEM_ID}/subaction
````````````````````

Executes a sub-action.

``````
/about
``````

Serves the glorious "about us" page.
